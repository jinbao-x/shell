#### 项目介绍
1.  shell_advanced为写好且已经编号的脚本，持续更新，用时请先在自己环境测试
2.  shell_basic为一些编写shell所需要注意的技巧性文档，可以坐地铁时瞄两眼


#### 使用说明

1.  脚本均以.sh结尾，且脚本的编译声明均为bash解释器的声明，可以直接下载使用
2.  脚本内均有注释，使用前请大致查看脚本具体内容与解释，避免因环境不一样导致脚本不适用
